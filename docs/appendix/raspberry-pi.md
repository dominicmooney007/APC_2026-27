# Appendix — Raspberry Pi 5

Everything in this course that touches the **Raspberry Pi 5** (a full Linux computer, programmed in Python via Thonny). The Whisplay HAT lessons also run on the Pi — see the [Whisplay appendix](whisplay.md).

## Lessons

| Lesson | What it covers |
|---|---|
| [Lesson 6 — Linux & remote access](../lessons/lesson-06-pi-intro.md) | Desktop, terminal, SSH/VNC/Pi Connect (optional skills) |
| [Lesson 7 — Python, Thonny & GPIO](../lessons/lesson-07-pi-gpio.md) | `gpiozero` LED output + button input |
| [Lesson 8 — ADC, I2C & SPI](../lessons/lesson-08-pi-buses.md) | MCP3008 analog input, bus devices |
| [Lesson 9 — Mini-project](../lessons/lesson-09-pi-project.md) | Log, plot & simple Flask web control |

Lesson 1's [electronics primer](../lessons/lesson-01-foundations.md) (breadboards, resistors, Ohm's law) underpins all Pi wiring.
