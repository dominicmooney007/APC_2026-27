# Appendix — Whisplay HAT

Everything in this course that touches the **PiSugar Whisplay HAT** — an add-on board that mounts on the Raspberry Pi 5's 40-pin header and adds an LCD, speaker, microphone, button(s), and RGB LED. Programmed in **Python on the Pi**, so these lessons build directly on the [Raspberry Pi track](raspberry-pi.md).

## Lessons

| Lesson | What it covers |
|---|---|
| [Lesson 10 — Meet the HAT & LCD](../lessons/lesson-10-whisplay-intro.md) | Mounting, driver install, hardware test, LCD from Python |
| [Lesson 11 — Buttons, LED & audio](../lessons/lesson-11-whisplay-io.md) | Button, RGB LED, speaker & mic (WM8960) |
| [Lesson 12 — Mini-project (game)](../lessons/lesson-12-whisplay-game.md) | Single-button game adapted from the driver examples |
