# Advanced Physical Computing 2026–27

Welcome! This site holds **every lesson, guide, and line of code** for Semester 1 of Advanced Physical Computing.

## How this course works

- **Schoology** is your front door — announcements, grades, and deadlines live there, with a link to each lesson here in order.
- **This site** is where you actually work: every lesson page has copy-pasteable code, wiring tables, and troubleshooting tips.
- **The GitHub repo** behind this site also holds all the code, organized by lesson in the `code/` folder.

!!! tip "Copy code the easy way"
    Every code block on this site has a **copy button** (top-right of the block). Use it — retyping code invites typos and indentation bugs.

## Getting the code

Two ways, both fine:

1. **Download ZIP** — on the repo page, click **Code → Download ZIP**.
2. **Clone with git** — in a terminal:

```bash
git clone https://github.com/REPLACE-USERNAME/apc-2026-27.git
```

Lesson 1 walks you through both.
