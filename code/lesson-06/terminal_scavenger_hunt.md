# Lesson 06 — Terminal Scavenger Hunt + Cheat Sheet

## The hunt (Checkpoint B)

From your home folder, using ONLY the terminal:

1. Make a folder `apc/lesson-06`
2. Create a file with your name: `echo "built by YOURNAME" > about.txt` (in that folder)
3. Prove it: `cat apc/lesson-06/about.txt`
4. Rename it to `signature.txt`, show a teacher `ls -l` of the folder

## Cheat sheet — keep this

| Command | What it does |
|---|---|
| `pwd` | Where am I? |
| `ls` / `ls -l` | What's here? (-l = details) |
| `cd folder` / `cd ..` / `cd ~` | Move in / up / home |
| `mkdir name` | Make a folder |
| `cp a b` | Copy a to b |
| `mv a b` | Move / rename |
| `rm file` | Delete — NO undo! |
| `cat file` | Show file contents |
| `echo "text" > file` | Write text into a file |
| `sudo apt update && sudo apt install X` | Install software |
| `hostname -I` | My network address |
| Tab | Complete the name (use constantly) |
| Up arrow | Previous command |
