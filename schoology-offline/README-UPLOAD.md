# Advanced Physical Computing — Offline HTML for Schoology

This folder holds a **self-contained, offline copy** of every APC lesson page.
Each `.html` file opens on any laptop with **no internet and no other files needed** —
all styling is baked in, fonts fall back to system fonts, and there are no CDN calls.

## What's here

- `index.html` — course home page
- `lesson-01-…` through `lesson-14-…` — the 14 lessons
- `arduino.html`, `raspberry-pi.html`, `whisplay.html` — appendix references
- `APC-Semester1-Offline.zip` — **all 18 pages bundled** for a single download
- `build inputs` live in `../docs/` (Markdown source); don't edit the HTML by hand

Every page has two download buttons near the top:

- **Download this lesson** — saves just that page (works fully offline, needs nothing else)
- **Download whole course (.zip)** — links to `APC-Semester1-Offline.zip`

## Uploading to Schoology

**Recommended — one resource per lesson (matches the "download to laptop" goal):**

1. In the course, go to the **Materials / Resources** area.
2. **Add File/Link/External Tool → File**, then upload each `.html` page you want as its
   own resource (e.g. attach `lesson-02-arduino-digital.html` to the Lesson 2 item).
3. Also upload `APC-Semester1-Offline.zip` once as a course-wide "Download the whole
   course (offline)" resource.

**Keep the per-page "Download whole course" button working:** that button uses a
*relative* link to `APC-Semester1-Offline.zip`. It only resolves if the zip sits in the
same place as the pages. On Schoology's individual-file hosting the relative link may not
resolve — so the reliable path is to also post the zip as its own resource (step 3). The
**Download this lesson** button always works regardless.

**Simplest alternative:** just upload `APC-Semester1-Offline.zip` as a single resource.
Students download it, unzip, and open `index.html` — every page and internal link works
because they're all sitting in one folder.

## Regenerating after you edit the lessons

Edit the Markdown in `../docs/`, then from the repo root (`apc-2026-27/`) run:

```bash
./rebuild-offline.sh
```

That rebuilds all 18 HTML files **and** refreshes the zip. First time only, install the
three Python packages it needs:

```bash
pip install markdown pymdown-extensions pygments
```

(Under the hood `rebuild-offline.sh` just runs `python3 build_offline.py docs schoology-offline`.)
